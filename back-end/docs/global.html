getAll(req, res)

Cette fonction récupère les artisans du mois dans la base de données. Elle va : - définir des constantes pour récupérer le mois & l'année de la date du jour, - récupérer les artisans de la table artisans du mois, en incluant les modèles des tables étrangères nécessaires, - filter les artisans en fonction du mois & de l'année de la date du jour.

sendFakeEmail(req, res)

Cette fonction envoie un faux e-mail. Elle récupère toutes les informations contenues dans le formulaire & les retourne dans la console. Elle va : - récupérer dans le corps de la requête tous les champs de l'e-mail, ainsi que l'artisan destinataire, - bloquer le nombre d'envois à un envoi toutes les 5 minutes, - créer un transporteur qui ne fera qu'afficher l'email au format JSON dans la console, - stocker tous les champs de l'email dans une constante (mailOptions).
